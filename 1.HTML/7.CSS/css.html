HTML'de CSS Kullanımı

CSS, HTML sayfalarında üç farklı şekilde kullanılabilir: inline, internal ve external.

1. Inline CSS

Inline CSS, doğrudan HTML elemanının style özniteliği ile uygulanır. Sadece ilgili elemana etki eder.

Bu kutu inline CSS ile stillendirildi.
<div style="..."> ... </div>

2. Internal CSS

Internal CSS, <style> etiketi ile <head> bölümünde tanımlanır. Sayfa içindeki birden fazla elemana etki edebilir.

Bu kutu internal CSS ile stillendirildi.
<style> .internal-demo { ... } </style>

3. External CSS

External CSS, ayrı bir dosyada yazılır ve <link> etiketi ile sayfaya eklenir. Büyük projelerde ve tekrar kullanılabilir stiller için idealdir.

Bu kutu external CSS ile stillendirildi.
<link rel="stylesheet" href="external-demo.css">

Özet

  • Inline CSS: Hızlı, sadece ilgili elemana özel stil.
  • Internal CSS: Sayfa içi, birden fazla elemana etki eder.
  • External CSS: Ayrı dosya, tüm siteye uygulanabilir, en çok tercih edilen yöntem.
